# URL Shortener 🔗

A full-stack URL shortener: paste a long link, get a short one back, and track
how many times it's been clicked. Built with a **FastAPI** backend, a
**React** frontend, and a **SQLite** database.

## Features

- Shorten any valid URL into a compact 6-character code
- Automatic redirect from the short link to the original URL
- Per-link click counter
- Clean, responsive single-page interface
- Interactive API docs auto-generated by FastAPI (`/docs`)

## Tech stack

| Layer     | Technology              |
|-----------|-------------------------|
| Frontend  | React + Vite            |
| Backend   | FastAPI (Python)        |
| Database  | SQLite (via SQLModel)   |

## Architecture

```
React frontend  ──HTTP──▶  FastAPI backend  ──▶  SQLite database
 (localhost:5173)          (localhost:8000)        (urls.db)
```

1. The React app sends the long URL to `POST /api/shorten`.
2. The backend generates a unique random code, stores the mapping, and returns it.
3. Visiting `GET /{code}` redirects to the original URL and increments the click count.

## API endpoints

| Method | Path               | Description                          |
|--------|--------------------|--------------------------------------|
| POST   | `/api/shorten`     | Create a short link                  |
| GET    | `/api/stats/{code}`| Get a link's target URL + click count|
| GET    | `/{code}`          | Redirect to the original URL         |

## Running locally

### 1. Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload       # runs on http://localhost:8000
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev                     # runs on http://localhost:5173
```

Open http://localhost:5173 in your browser.
